## 📖 Overview
This project is a **Spring Boot REST API** designed to manage:  

✅ **Student Profiles** (CRUD operations)  
✅ **Tasks** with status tracking (`TODO`, `IN_PROGRESS`, `COMPLETED`, `BLOCKED`)  

It demonstrates clean **Controller-Service-Model** architecture with **in-memory storage (HashMap)** for simplicity.  

---

## ✨ Features

### 🎓 Student Profile Management
- ➕ Create a Student → `POST /studentprofile/create`  
- 📖 Get Student by ID → `GET /studentprofile/{studentId}`  
- ✏️ Update Student → `PUT /studentprofile/update/{studentId}`  
- ❌ Delete Student → `DELETE /studentprofile/delete/{studentId}`  

### 📌 Task Management
- ➕ Create Task → `POST /tasks/create`  
- 📖 Get Task by ID → `GET /tasks/{id}`  
- ✏️ Update Task → `PUT /tasks/update/{id}`  
- ❌ Delete Task → `DELETE /tasks/delete/{id}`
